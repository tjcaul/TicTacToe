#TicTacToe
*By tjcaul (Tyson Caul)*

Command line usage:

java TicTacToe <boardsize> [players> [<ais> [auto]]]

**Examples**

java TicTacToe 5            //5x5 game, human vs human

java TicTacToe 3 XA         //3x3 game, human "X" vs human "A"

java TicTacToe 3 X j        //3x3 game, human "X" vs computer "j"

java TicTacToe 3 "" xo      //3x3 game, computer vs computer

java TicTacToe 3 "" xo auto //3x3 game, computer vs computer, plays again
                            //without prompting
